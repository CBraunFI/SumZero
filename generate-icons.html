SumZero PWA Icon Generator

This tool converts the SVG icon to PNG files in different sizes.

Instructions:

  1. Open this HTML file in a browser
  2. Download all the generated PNG icons
  3. Save them to the public/icons/ folder
  4. Make sure the filenames match the manifest.json entries

Required files:

  • icon-16x16.png
  • icon-32x32.png
  • icon-72x72.png
  • icon-96x96.png
  • icon-128x128.png
  • icon-144x144.png
  • icon-152x152.png
  • icon-192x192.png
  • icon-384x384.png
  • icon-512x512.png
